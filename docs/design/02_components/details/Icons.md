# Icons (Voice Icon / number)

## 概要
アイコンや単一要素として利用されるコンポーネント群。

## 🎨 構造と事実データ

### `Voice Icon` (音声波形アイコン)
* **構成データ**: 9つの `Rectangle`
  * **適用カラートークン**: `On Surface`
  * （※アイコンの幅や角丸はトークン化されていない独自のマイクロスタイルで形成）

### `number` (ステップ数字アイコン)
* **適用テキストトークン**: `text-headline`
* **適用カラートークン**: `On Surface`
