# momeo プロダクトオーバービュー

## momeo とは
momeo は、音声入力を第一に考えた「音声ファースト」のメモアプリケーションです。ユーザーがアプリを開いた瞬間から、できるだけ早くリスニングを開始し、シームレスに音声をテキストメモとして記録することを目的としています。

音声認識はオンデバイス STT（`sherpa_onnx` + NeMo CTC）で行い、モデルが端末にあれば実行は完全オフラインです。ユーザーに許可を求める権限はマイクのみです。

## アプリ全体の大まかな振る舞い
アプリは起動後、必要な権限を速やかに取得した上で、すぐに音声入力が可能な状態へと遷移します。ユーザーは録音開始ボタンを探す必要なく、そのまま話し始めることができます。

## メインアプリフロー
1. **アプリ起動**: ユーザーがアプリアイコンをタップ。
2. **スプラッシュシーケンス**: 起動時にアプリのコンセプトを伝える短いアニメーションテキストを表示。初回起動時はフルシーケンス、2回目以降は短縮表示。
3. **権限確認**: 必要な権限（マイクのみ）が許可されていない場合、権限を要求。
4. **準備ゲート**: リスニング直前に文字化エンジンの準備完了を確認。準備済みなら素通りし、未完了（Android 初回のモデルダウンロード中など）なら待ち画面で足止めして完了後に自動で進む。
5. **リスニング画面**: メイン画面。アプリは直ちにリスニング状態となり、ユーザーの発話を待ち受ける。
6. **メモカード作成**: ユーザーが話し始めるとリスニング中インジケーター（アクティブカード）が先頭に現れ、発話が確定すると、テキスト化されたメモカードとして画面に蓄積される。
