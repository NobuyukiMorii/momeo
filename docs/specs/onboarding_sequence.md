# オンボーディングシーケンス仕様 (旧: スプラッシュ)

## 概要
このシーケンスは、**アプリをインストールした後の初回起動時に1回だけ表示される**単一の画面です。2回目以降の起動ではスキップされ、直接次のフロー（権限確認またはリスニング画面）へ進みます。
複数の画面遷移ではなく、1つの画面内でアプリのコンセプトを伝えるテキストがアニメーションしながら切り替わります。

## アニメーションの挙動
- 最初のテキストは初期状態から画面左下に表示されます。
- 一定時間経過後、次のテキストが右からフェードインし、所定の位置で停止します。
- 同時に、現在のテキストは左に移動しながらフェードアウトします。
- この遷移が設定されたテキストの数だけ繰り返されます。

## 表示テキスト例（Figma参照）
現在のデザインドキュメントで示されているテキスト項目は以下の通りです。
1. `momeo`
2. `Open. Speak. Saved.`
3. `Auto-start`
4. `Auto-stop`

## 設定可能な項目
この画面は柔軟な設定が可能なテキストシーケンスとして実装されるべきです。以下の項目は固定せず、変更可能にします。
- テキスト項目の数
- 各テキストの内容
- アニメーションの持続時間
- 各テキストの停止時間
